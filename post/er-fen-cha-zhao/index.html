二分查找

问题

  • 给定一个已排序的数组，要求用二分查找找到指定元素的索引，找不到返回-1。

思路

  • 1 设置下标为 low，上标为high，中间数据为mid = （low+high）/2；

  • 2 当目标target小于 中间mid 元素时 ，low = mid，mid 重置为（low+high）/2；

  • 3 当目标target大于 中间mid 元素时 ，high = mid，mid 重置为（low+high）/2；

  • 4 重复步骤2，3至找到目标taget ，返回mid；或者 到 low >=high, 没有找到 ，返回-1；
    ###代码实现

      public static int  Binarysearch(int target,int[] array) {
          int low=0;
          int high = array.length;

          while (low < high){
              int mid = (low+high)>>1;
              if(target == array[mid]){
                  return mid;
           }
              else if(target<array[mid]){
                  high = mid;
              }else {
                  low = mid;
              }
       }
      return -1;
  }
