冒泡排序

2023-03-01 算法

2023/03/01

问题

  • 给定一个数组，要求用冒泡法将其从小到大重新排序。

思路

  • 冒泡即是利用了在水中重的元素下沉，轻的上浮。
  • 从第一个元素开始，跟后一个元素比较，大于则交换位置，否则位置不变。
  • 这样每轮排序找到最大的一个元素，放到后面。
    ###代码实现
public  static  void BubbleSort1(int [] list){
   int max = 0;
   for (int j = 0; j < list.length ; j++) {
       int flag = 0;
      for (int i = 0; i < list.length - j-1; i++) {
           if (list[i] > list[i + 1]) {
              max = list[i];
                list[i] = list[i + 1];
              list[i+1]=max;
               flag++;
           }
      }
     if(flag == 0 ){
         break;
   }
}

}

  • 接收一个数组参数, 双重for循环，第一层是每轮查找，第二层是每次比对大小。 flag = 0 , 一轮查找中发生元素交换则自增，若一轮查找都没发生交换，数组已经是有序状态，直接返回。
